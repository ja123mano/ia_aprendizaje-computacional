# Instrucciones
----------------
Para poder obtener resultados favorables del modelo de clasificación, abrir google earth en pantalla completa
y esconder la barra lateral y las barras de herramientas inferiores así como superiores. De igual forma, ocultar
todos los íconos, etiquetas y demás añadidos sobre las imágenes satelitales para obtener mejores resultados a la
hora de correr el programa.

# Ejecución
------------
Este archivo abre una ventana la cual contiene una etiqueta que indica el bioma predicho por el modelo clasificador
y un botón para realizar el screenshot.

Los pasos a seguir son los siguientes:
1. Abrir google earth y seguir las instrucciones descritas arriba
2. Ejecutar el archivo "ui.py"
3. Elegir el bioma a predecir dentro de google earth
4. Presionar el botón de la interfaz y ver la predicción
5. Repetir pasos 3 y 4 hasta el cansancio

# Información
--------------
- El modelo de clasificación usado es una red neuronal de 1 capa oculta con 100 neuronas.
- La exactitud de este modelo es de apenas un 55% más o menos.
- La predicción no siempre será la correcta debido a los pocos descriptores por imagen usados para el entrenamiento
del modelo
